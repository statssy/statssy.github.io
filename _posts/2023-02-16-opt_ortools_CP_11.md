Google OR-Tools Routing (8)Penalties and Dropping Visits에 대한 간단한 정리

---
  
### 불이익 및 방문 감소
- 제약조건으로 인해 실현 가능한 솔루션이 없는 라우팅 문제를 처리하는 방법 설명
- 예를 들어 모든 위치의 총 수요가 차량의 총 용량을 초과하는 경우 솔루션은 없음
- 이런 경우 차량은 일부 수요가 있는 위치 방문 기록을 삭제해야함
- 근데 어떤 방문 기록을 삭제할지를 결정해야함
- 이 문제를 해결하기 위해서는 모든 위치에 새로운 비용(패널티)를 적용해야함
- 위치 방문이 누락될때마다 총 이동거리에 패널티가 추가
- 그런 다음 총 거리와 패널티를 최소화 하는 경로를 찾음
  
### 패널티의 크기
- 모든 노드의 합계를 초과하는 패널티를 준 경우 : 한 엣지를 지우고 더 이상 지우지 않는다. 패널티가 크니까
- 모든 엣지를 갈 필요가 없는 경우에는 패널티를 작게 하면된다. 그러면 추가적으로 엣지를 더 지우고 경로를 만든다.

### 용량 제약조건 및 패널티 추가
- 수요 콜백 및 용량 제약조건을 추가하고 ```AddDisjunction``` 메서드를 사용하여 패널티 추가
- 예시에서는 위치마다 동일한 패널티를 줬지만 각각 다른 패널티를 추가할수 있음

def main():
    """Solve the CVRP problem."""
    # Instantiate the data problem.
    data = create_data_model()

    # Create the routing index manager.
    manager = pywrapcp.RoutingIndexManager(len(data['distance_matrix']),
                                           data['num_vehicles'], data['depot'])

    # Create Routing Model.
    routing = pywrapcp.RoutingModel(manager)


    # Create and register a transit callback.
    def distance_callback(from_index, to_index):
        """Returns the distance between the two nodes."""
        # Convert from routing variable Index to distance matrix NodeIndex.
        from_node = manager.IndexToNode(from_index)
        to_node = manager.IndexToNode(to_index)
        return data['distance_matrix'][from_node][to_node]

    transit_callback_index = routing.RegisterTransitCallback(distance_callback)

    # Define cost of each arc.
    routing.SetArcCostEvaluatorOfAllVehicles(transit_callback_index)


    # Add Capacity constraint.
    def demand_callback(from_index):
        """Returns the demand of the node."""
        # Convert from routing variable Index to demands NodeIndex.
        from_node = manager.IndexToNode(from_index)
        return data['demands'][from_node]

    demand_callback_index = routing.RegisterUnaryTransitCallback(
        demand_callback)
    routing.AddDimensionWithVehicleCapacity(
        demand_callback_index,
        0,  # null capacity slack
        data['vehicle_capacities'],  # vehicle maximum capacities
        True,  # start cumul to zero
        'Capacity')
    # Allow to drop nodes.
    penalty = 1000
    for node in range(1, len(data['distance_matrix'])):
        routing.AddDisjunction([manager.NodeToIndex(node)], penalty)

    # Setting first solution heuristic.
    search_parameters = pywrapcp.DefaultRoutingSearchParameters()
    search_parameters.first_solution_strategy = (
        routing_enums_pb2.FirstSolutionStrategy.PATH_CHEAPEST_ARC)
    search_parameters.local_search_metaheuristic = (
        routing_enums_pb2.LocalSearchMetaheuristic.GUIDED_LOCAL_SEARCH)
    search_parameters.time_limit.FromSeconds(1)

    # Solve the problem.
    assignment = routing.SolveWithParameters(search_parameters)

    # Print solution on console.
    if assignment:
        print_solution(data, manager, routing, assignment)


if __name__ == '__main__':
    main()
